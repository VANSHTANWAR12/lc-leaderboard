About

This small project displays a friends leaderboard using public LeetCode profiles. Add LeetCode usernames and the app will fetch solved counts (Easy / Medium / Hard) and rank friends by total solved problems.

Note: The app scrapes data from LeetCode's GraphQL endpoint. If a username cannot be fetched, counts will show as 0.

If you'd like, I can add features: profile links, remove friend, per-day stats, or GitHub Actions to auto-refresh.
